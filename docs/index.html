Two views, one dataset

The Trophies page is the social view — a flat list of every bug we've shipped upstream, with status and a link to the original report. It mirrors the style of the AFL++ trophy case.

The Benchmark page is the research view — the same bugs framed as an evaluation harness for fuzzers, agentic vulnerability discovery systems, and triage tools, in the spirit of ExploitBench / V8-bench. Each PoC is scored against a 4-flag capability ladder (reach / crash / class / site) by deterministic oracles, with 3-round unanimity.

Both pages read from the same bugs.json — one source of truth. Per-bug build artifacts, the Go MCP server, and the Python runner live in the source repo.
